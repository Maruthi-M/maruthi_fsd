Java 8 Features

Following list is a highlight of important features, there are other minor enhancements, security features, bug fixes are available as part of Java 8.

Lambda Expressions, Pipelines and Streams, Date and Time API, Default Methods, Type Annotations, Nashorn JavaScript Engine, Concurrent Accumulators, Parallel operations, PermGen Space Removed, TLS SNI.
